closeMagic The Gathering

Este site foi um dos primeiros que iniciei, foi um projeto pessoal que ainda está em desenvolvimento. Ao perceber que vender minhas cartas de Magic The Gathering (um trade carding game), eu tinha muita dificuldade, devido ao fato de que ao levar as cartas físicas para que as pessoas vissem, havia desorganização das cartas, riscos de estragá-las ou mesmo risco de furto das mesmas. closeJogo Pong

Este site foi desenvolvido como atividade ao decorrrer da minha graduação em Análise e Desenvolvimento de Sistemas na matéria de Programação de Scripts ministrada pelo professor Luciano Pelissoli. No presente projeto, deveríamos, primariamente criar um jogo de Pong utilizando o Javascript, e, se conseguíssemos a tempo, desenvolver uma aparência melhor com o HTML e CSS.

Dentro dos requisitos do projeto, o principal envolvia que ambas as raquetes se movessem independes, de acordo com o apetar das teclas "w" e "s" para o player 1, e "up" e "down" para o player 2; a bolinha deveria se mover sozinha de forma aleatória, sem ultrapassar os limites de campo e jogadores, alterando a direção de seu deslocamento ao colidir com um dos respectivos limites; deveria também, no caso de a bolinha colodir com os limites horizontais do campo, que a pontuação fosse contada em um placar.

Este site foi desenvolvido como atividade ao decorrrer da minha graduação em Análise e Desenvolvimento de Sistemas na matéria de Programação de Scripts ministrada pelo professor Luciano Pelissoli. Neste projeto, deveríamos criar uma página HTML que acessasse uma rotina externa, neste caso o ViaCEP, fazendo acesso pelo Javascript ao site, para verificar: se o CEP inserido é válido, e, se sim, retornasse as informações de logradouro, bairro, cidade e estado. Cardápioadd

Em breve no GitHub

closeCardápio

Este projeto foi desenvolvido durante meu curso de Desenvolvimento Front-End ministrado pelo professor Paulo henrique Leme Ramalho na FATEC de Bragança Paulista. Neste código, a ideia era que nos familiarizássemos com o Materialize e praticássemos os códigos HTML e CSS, e para tanto, foi-nos proposto criar um cardápio, podendo ser simples para mesas de estabelecimento, ou mais desenvolvidos para um e-comerce. Nesta ideia, desenvolvi um cardápio de uma doceria sob demanda, onde nos cardápios, poderia ser selecionada as opções para a construção personalizada de seus bolos verificando os preços de cada componente, assim como a possibilidade de montar seu "cento" de doces para festas de acordo com o preço da dezena. Essa ideia permitiria a compra de uma variedade maior de sabores a cada cento, além claro da possibilidade de adicionar ao carrinho, os doces unitários se assim o cliente desejar.

Com o caminhar da minha graduação, pretendo retornar ao projeto para desenvolvê-lo melhor, e adicionar as funções Back-End para o site.

Xadrezadd

Em breve no GitHub

closeXadrez

Este projeto é um projeto particular, onde, para praticar a linguagem, decidi fazer um jogo de xadrez. Em cada "rodada", deve ser selecionada a peça a ser movida de acordo com sua posição no tabuleiro, se a peça selecionada não for do jogador atual, não é permitida a movimentação, e se a peça for válida, deve-se então entrar com a posição do tabuleiro que a peça irá. Dentro de cada movimentação há a verificação de: qual peça foi selecionada, as movimentações que a peça pode fazer, se a movimentação selecionada pelo jogador é permitida, se haverá a remoção de outra peça no tabuleiro, e se a movimentação colocou o próprio rei, ou o rei adversário em cheque. Devido a complexidade deste projeto, ele ainda está em desenvolvimento.
